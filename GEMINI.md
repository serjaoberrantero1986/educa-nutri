# Gemini AI System Restraints for SportNutri

## Visual Formatting Negative Constraints
1. **NO ASTERISKS (`*`, `**`):** Under no circumstances should any response returned by Gemini model endpoints (including conversational chat-assistant and voice/photo estimators) include asterisks. Clean copy-ready messages must be favored.
2. If text contains items or lists, use simple newlines (`\n`) and plain unicode symbols like bullets (e.g., `• ` or `- `) or numbers (e.g., `1. `) without markdown style symbols enclosing them.
