# Project Guidelines for SportNutri Coding Assistants

## Clean Typography Constraints (No Bold/Emphasis Overuse)
1. **Never use Asterisks in Bot/Assistant Message and Response Output:** All user-facing messages generated by the Nutri-Assistant bot, Gemini AI endpoints, or custom text formatters must be completely free of asterisks (`*` or `**`). Double asterisks for bolding or single asterisks for list item marks must be completely stripped or avoided. Use simple human-readable formatting, plain text, and line breaks instead.
2. Ensure any text responses rendered inside the `NutriAssistant` component do not contain markdown symbols. If any are detected, sanitize them programmatically via `.replace(/\*/g, "")` or similar regexes to preserve a clean and polished UI workspace.
